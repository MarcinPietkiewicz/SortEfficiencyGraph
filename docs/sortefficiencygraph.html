Sort Efficiency Graph module¶

class sortefficiencygraph.SortEfficiencyGraph[source]¶

Bases: object

This class is used to display on graph time complexity of two sorting algorithms. It takes 2 algorithms and creates 4 random lists to sort with given number of elements. Then it measures time of each algorithm when sorting those random lists. Lastly it displays graphs comparing time complexity of two sorting algorithms.
